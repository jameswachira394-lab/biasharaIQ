## ❓ Common Questions

### Where do I deploy?
- **Frontend**: Vercel (Next.js optimized)
- **Backend**: Render (Python optimized)
- **Database**: Render PostgreSQL (auto-provisioned)

### How do I set environment variables?
- **Render**: Use Render dashboard → Environment
- **Vercel**: Use Vercel dashboard → Environment Variables

### How do I verify it's working?
- Run: `./verify-deployment.ps1`
- Check: `curl https://biasharaiq.onrender.com/health`
- Test: Login flow in browser

### What if something breaks?
- Check deployment logs (Render/Vercel dashboard)
- See [DEPLOYMENT.md](DEPLOYMENT.md) Troubleshooting section
- Use rollback feature in Render/Vercel

### Can I still develop locally?
- Yes! Development unchanged
- Backend: `http://localhost:8000`
- Frontend: `http://localhost:3000`
